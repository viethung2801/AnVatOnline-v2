## AnVatOnline
Spring boot + thymeleaf + SQL server<br>
Main features:<br>
• Login, Register (Spring Security)<br>
• Order online<br>
• POS (Point of Sale)<br>
• Manage products, customers, orders, carts, categories.<br>
• Statistics<br>
